## 👾 Activity 10👾
I followed the demo to make a simple sketch that allows users to change the color, shape and size of the object. Then, I included a pattern amount function, which creates a radial pattern as the number of shapes increases. To easily reset the sketch, I added a reset button that resets the number of shapes back to 8 and the color back to a dark gray.
