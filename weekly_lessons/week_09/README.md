## 👾 Activity 8A👾
We followed the demo code and installed the physical components according to the diagrams. There were 3 portions to the activity, leaving the LED to be always on, controlling the on and off using Arduino IDE and letting the light fade on and off. 

## 👾 Activity 8B👾
For this activity, we tested out three different sensors: the phototransistor sensor, the force sensor and the ultrasonic sensor. We found the ultrasonic sensor to be the most sensitive to change, resulting in more accurate results.
