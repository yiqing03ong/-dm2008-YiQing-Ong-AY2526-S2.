## 👾 Activity 2B👾
I referenced the grid code demonstrated in class, then experimented with the colors. I then included for loops to alternate between circles and squares in my grid pattern. To make the pattern more dynamic, I used the modulo operator to control both the shape placement and color variation. 

I also introduced randomness in the fill colors for certain shapes to create variation and a more organic visual effect. Lastly, I added an interactive element using mousePressed(), where clicking the canvas toggles an invert variable, allowing the pattern to switch dynamically and making the sketch more engaging.

Link to sketch: https://editor.p5js.org/yiqing03ong/sketches/cTgQUsGbg
