## ✨Astronomer's Respite🌟
For my final project, I wanted to create a more contemplative and calm project, where someone can just rest their minds after a tiring day. I wanted this to link to my final year project, which is going to be an interactive graphic novel about a lost jellyfish who is guided back home by the moon and stars. Hence, I thought it would be nice to recreate a scene from my graphic novel.  

This interactive sketch allows the user to sit and watch the twinkling stars, directly interact with the night sky by tapping and dragging the stars. By tapping stars, you form connections, and by dragging the stars around, you can form your own constellations. With enough creativity, one can even draw simple characters with this sketch. 

For this project, I made use of the basic concepts learned in this module. I created the stars using arrays, and used a for loop to populate the sky with 150 stars, with random positions and slightly varying sizes. To form the constellation lines, I calculated the distance between two stars, and let maxDist = min(width, height) * 0.15. This way, if the user wants two stars to connect, they need to ensure the stars are close enough. 

Initially, I hadn't thought about making this sketch out to have a use in drawing things. The project started off by just aiming to allow the users to draw simple constellation patterns, but somewhere along the line, I ended up discovering that my constellations can have multiple connections to stars, forming a web, which allows for more complex drawing. My code also allows users to repeatedly tap on previously tapped stars to thicken the lines of the constellation, which can allow for more complex compositions. I also included a function for users to delete their constellations by pressing Q.
