## 👾 Activity 4A👾
I created 3 cookie flavours: chocolate, vanilla and strawberry, and gave each cookie randomized chocolate chip placements each time the sketch is started up. I added interactivity by allowing the user to move all three cookies together using arrow keys, using keyIsDown, which allows the user to more seamlessly move around the cookies. To add to the experience, I drew a simple oven background. 

Link to sketch: https://editor.p5js.org/yiqing03ong/sketches/XfjlxEYB3 

## 👾 Activity 4B👾
I created moving agents that have random starting positions, sizes and velocities. I used a semi-transparent background to create a motion trail to make it look more interesting. When they move beyond the canvas, they wrap around to the opposite side as well. To make the composition even more dynamic, I added a size-changing effect using a sine function, which makes each agent pulse and shrink. I also added more interactivity with the mouse, as the agents repel away from the cursor and change their moving directions. 

Link to sketch: https://editor.p5js.org/yiqing03ong/sketches/JD6HuK5_n
