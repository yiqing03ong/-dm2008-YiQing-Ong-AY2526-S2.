## 👾 Reflection 👾

**🤖My experience with coding prior to this course🤯** 
Like basically none. Before this, I had taken a module in game design, but all my code was basically ChatGPT'ed. Plus, we were using C# to create our games on Unity, so I was entirely foreign to JavaScript. In the beginning of this module, I was really intimidated by the content and wasn't sure if I would be able to catch up. 


**🌱After DM2008☘️**
I still can't say I am a coding savant, but after a whole semester of trying at failing and trying again in p5.js, my understanding on computer language has definitely deepened, and I have gained, albeit only a little bit, of confidence in coding. Overall, there is still alot for me to learn, and this is not the end of my coding journey, but as of right now, I am proud of the tiny progress I've made! 


**🌷My favourite part of this experience💖**
I really enjoyed being in a class of talented individuals and seeing their creative visions come to life in their sketches. Seeing their work gave me inspiration and motivation to do better and really made me appreciate how much I can learn from others in a collaborative creative environment.
